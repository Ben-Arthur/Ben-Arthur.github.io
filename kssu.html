Collision-box data structure (WIP!)

Example: The game uses a 4 byte pointer at memory address 0x0207703C to get data for Kirby's current hitbox.

Offset Description
0x00 0000 (always?)
0x02 Hitbox frame duration
0x03 Flag for end(01)/repeat(04)/continue(00)? once timer reaches 0
0x04 Hitbox Damage (signed)
0x06 unknown. Possibly hitbox ID?
0x07 unknown
0x08 2 flags? Seems to be between 0 and 2
0x0A Effect (normal, can't hit bosses, fire, ice, electric, etc.)
0x0B Strength (related to length/direction of knockback animation)
0x0C unknown
0x0D unknown (always 0?)
0x0E unknown. Possibly hitbox group ID???
0x0F unknown. Usually 0x0, but 0xFF for hitboxes with no damage
0x10 Offset of the collision-box's center point (x,y)
0x14 Offset of the top left from the center point (x,y)
0x18 Offset of the bottom right from the top left
